Cargo

```bash
cargo --version
cargo new hello_cargo

cargo build
./target/debug/hello_cargo
cargo run # build & run

cargo check # does not build exec
cargo build --release # debug by default

cargo update # update dependencies (crates)

cargo doc --open
```
